# Serial Killer JavaFX Application

***

Information shown is based on real data from [Serial Killers Database](http://murderpedia.org) with some other information filled in. 

Evolving a class project that was origincally built with PHP/HTML and transforming it into a JavaFX application. All information is pulled from a mySQL database that will live on Amazon Web Service's RDS. 

Uses Model-View-Controller software design pattern. Database schema was designed from scratch.
